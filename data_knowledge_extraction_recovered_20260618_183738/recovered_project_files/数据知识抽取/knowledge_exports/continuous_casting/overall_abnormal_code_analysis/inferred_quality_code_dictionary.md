## 建议使用方式

1. 把 `品质异常代码1-5` 先拉成长表，不要当成五个固定缺陷标签。
2. 用上表的“推断缺陷/异常方向”作为临时语义标签。
3. 对置信度为“中/中低”的代码，必须结合现场码表或人工样本复核。
4. 建模时可先合并成几个大类：液面/卷渣风险、强过程波动、拉速-塞棒流量波动、热交换不均、交接段/中间包过渡、温度/过热度、覆盖剂/保护渣工况。
