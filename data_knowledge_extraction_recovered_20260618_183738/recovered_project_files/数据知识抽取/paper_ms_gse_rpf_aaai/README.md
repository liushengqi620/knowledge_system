# MS-GSE + RPF AAAI Paper Draft

This directory is the manuscript workspace for the proposed AAAI paper.

Official template status checked on 2026-06-19:

- AAAI-26 conference page links an `AAAI-26 Author Kit`.
- The AAAI 2026 Overleaf template states that papers must use `aaai2026.sty` and `aaai2026.bst`.

The files `aaai2026.sty` and `aaai2026.bst` are not vendored here yet because the official AAAI author-kit endpoint returned HTTP 403 in this environment. Download them from the official AAAI-26 Author Kit before final compilation.

Current files:

- `main.tex`: English AAAI-style LaTeX manuscript skeleton.
- `references.bib`: bibliography entries for the current method story.
- `main_zh.md`: Chinese paper skeleton for discussion and translation alignment.

The manuscript intentionally uses placeholders for SOTA tables until full-budget experiments finish. Do not claim the smoke results as final paper evidence.
